File structure:

To extend:
All content lives in shared/config.js — no other file needs to be touched for content changes.
To add backend persistence, replace localStorage calls in student/index.html with fetch() calls to Supabase, Firebase, or your own API.
To add auth: wrap startSession() with Firebase Auth, Auth0, or Supabase Auth.
